# 🏦 DevOps Banking Web Application

## 📌 Project Overview
This project demonstrates a DevOps pipeline using a Flask web application.

## 🚀 Features
- Deposit money
- Transfer money
- Transaction history
- Clean UI dashboard

## ⚙️ Technologies Used
- Python (Flask)
- Docker
- GitHub
- GitHub Actions (CI/CD)

## 🔄 DevOps Workflow
1. Code pushed to GitHub
2. CI/CD pipeline triggered
3. Docker image built
4. Application tested automatically

## 🐳 Run Locally
docker build -t flask-devops-app .
docker run -p 5000:5000 flask-devops-app

## 🌐 Access App
http://localhost:5000
